## 操作方法
- 音声認識※もしくは「Space + Enter」で正解判定！
- 他のボタンを押すと「✖」が表示されます！
- カメラが複数ある場合はCボタンで切り替えが可能です（説明シーンのみ）

※音声認識はWindowsのみで利用可能です．
